# Financial Document Analysis Using OCR

## Project Overview
This project extracts useful information from invoice documents using OCR and rule-based text processing.  
It converts unstructured invoice text into structured data and validates financial totals.

## Features
- OCR text extraction from invoice files
- Text cleaning and normalization
- Invoice data extraction (bill number, date, totals)
- Validation of subtotal + tax against grand total

## Tech Stack
- Python
- Tesseract OCR
- Regular Expressions (Regex)
- Git & GitHub
